# Name

- The app is called **compass.tne.ai**

# Users

- Users are **Technical Managers** and **Software Engineers** who need specialized, expert analysis of complex codebases. They are highly technical and expect realistic, accurate, and actionable feedback.

# Value proposition

A secure, micro-transaction-based marketplace that connects developers with highly specialized AI expert agents (starting with a C++ Expert) to provide actionable, premium code reviews via seamless cryptocurrency payments.

# Key features

A streamlined web platform with an integrated agent-to-agent (A2A) payment pipeline, including:
- **Marketing & Landing Page:** Advertises the specific expertise of available agents (e.g., C++ Expert).
- **Salesbot Agent:** An automated chatbot that acts as a gatekeeper, clarifying the user's intent and determining if their problem aligns with the platform's capabilities (e.g., rejecting Python requests if only C++ is supported).
- **Expert Review Engine (C++ Agent):** Analyzes pasted code snippets to determine quality. Returns a partial document/review to prove credibility.
- **Micro-transaction Paywall (X.402):** A crypto-based payment gateway operating on a testnet that unlocks the full expert review.
- **Solid Vault Receipt:** A secure, transparent receipt generated upon successful payment.
- **Full Report Delivery:** Provides the user with a comprehensive, annotated document containing recommendations for their codebase.

# Example scenario

Here is an example session based on our proposed MVP flow:

- Alice, a Technical Manager, is struggling with a complex C++ module and navigates to the compass.tne.ai homepage.
- She interacts with the automated Salesbot, explaining her problem. The Salesbot verifies her request is related to C++, confirms the platform can help, and directs her to input her code.
- Alice pastes her C++ code snippet into the engagement interface.
- The C++ Expert Agent performs an initial analysis. It generates a "teaser" response—perhaps noting that the code has serious memory leak vulnerabilities but withholding the exact solution.
- Convinced of the agent's expertise, Alice encounters the paywall. She initiates an X.402 micropayment request via a crypto testnet.
- The transaction clears. Alice is issued a Solid Vault Receipt and an A2A card confirming the engagement.
- The platform unlocks and delivers the final, comprehensive technical document with the exact C++ recommendations and fixes.
